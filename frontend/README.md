# Sistem Informasi Manajemen Inventaris Barang Kantor Berbasis Web

## Deskripsi
Website manajemen inventaris barang berbasis web full-stack menggunakan Python Pyramid (backend) dan React JS (frontend).

## Fitur
- CRUD Barang
- CRUD Kategori
- Autentikasi Dasar
- UI Responsif
- Integrasi API via Axios

## Dependensi
### Backend
- Python 3.9+
- pyramid
- sqlalchemy
- psycopg2-binary

### Frontend
- react
- react-router-dom
- axios
- tailwindcss / bootstrap (opsional)
